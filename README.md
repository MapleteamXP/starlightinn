# 🌟 Starlight Inn — Desktop Edition

**Just double-click and play.** No internet required. No accounts. No hosting headaches.

---

## 🚀 How to Play (Choose Your Path)

### Option A: Double-Click the HTML (Fastest — No Server Needed)
1. Double-click `index.html`
2. Your browser opens
3. Enter your name → pick a character → play!

✅ Works completely offline  
✅ All features available (avatars, areas, trading, chat, island building)

---

### Option B: Use the Launcher (Starts a Local Server)
1. Double-click **`Start Starlight Inn.bat`**
2. A nice console window opens and starts the server
3. Your browser opens automatically to `http://localhost:8080`
4. Play!

✅ Better for sharing with friends on your local network  
✅ The server shows a cool ASCII banner

**Requirements:** Node.js installed (get it free at https://nodejs.org)

---

## 🎮 What's Inside

| File | Purpose |
|------|---------|
| `index.html` | The entire game in one file (self-contained) |
| `server.js` | Local server for the launcher |
| `Start Starlight Inn.bat` | Windows launcher — double-click to start |

## 🌐 Want to Publish Online Later?

Click the **🌐 globe button** in the game's top bar. It opens a panel with one-click links to:
- **Netlify Drop** — drag & drop, instant URL
- **Tiiny.host** — single file hosting
- **GitHub Pages** — free forever hosting

Or manually:
1. Zip the `index.html` file
2. Go to https://app.netlify.com/drop
3. Drop the zip → get a live URL

## 🖼️ Game Features

- 🏨 **6 Areas:** Starlight Inn Lobby, Sunset Beach, Whisperwood Forest, Canopy Treehouse, Blossom Park, Your Island
- 🎭 **6 Characters:** Neko (catgirl), Rasta, Luna (mage), Cpt. Kiko (pirate), Bun-Bun (bunny), Byte (robot)
- 🎨 **Color Customization:** Skin (8 tones), Hair (10 colors), Clothes (10 colors)
- 💱 **Trading System:** Rarity tiers with glowing borders — Rare, Very Rare, Almost Unique, Unique, LTD
- 💬 **Chat:** Talk to AI-controlled players and NPCs
- 🏝️ **Island Builder:** Place furniture on your personal island
- 🎁 **Daily Rewards:** Claim login bonuses
- 📱 **Mobile Friendly:** Touch controls work on phones and tablets

---

## 🛠️ Tech Stack

- **Frontend:** Vanilla HTML5 + Canvas 2D (no frameworks)
- **Server:** Node.js (optional, for local hosting)
- **Art:** Procedural pixel-art rendered live on Canvas
- **Size:** ~50KB single file — fits on a floppy disk
